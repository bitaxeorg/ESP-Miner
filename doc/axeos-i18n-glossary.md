# AxeOS i18n Glossary (Authoritative)

This glossary standardizes terminology for the AxeOS web UI. Use these terms consistently across all translations.

## Tone and style
- German (de): informal Du.
- Spanish (es): informal tu.
- Keep labels short; UI space is tight.
- Keep units unchanged: `°C`, `W`, `V`, `MHz`, `GH/s`, `TH/s`.
- Mining jargon should remain familiar to miners.

## Global rules
- Keep the word `Stale` unchanged in all languages: EN/DE/ES = `Stale`.
- Keep `Shares` unchanged in all languages: EN/DE/ES = `Shares`.
